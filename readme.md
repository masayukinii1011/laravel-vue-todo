# README

簡易的なTodo機能を作りました。<br>バックエンドはLaravel、フロントエンドはVue.jsで実装。<br>バックエンドとフロントエンドのデータの受け渡しはaxiosライブラリを使用してAPIで行う。<br>Vue.js単一コンポーネントファイル内のtemplateはpug、styleはscssで記述。<br>Vue RouterでSPA化。<br>Herokuへデプロイ。

課題：ログイン認証、レスポンシブ未対応
